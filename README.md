# 🛡️ Network Scanner

A Python-based network reconnaissance tool designed to scan target systems for open ports and identify active services during basic security assessment tasks.

---

## 🎯 Purpose

This tool was built for cybersecurity learning and practical understanding of:

- Network scanning fundamentals
- TCP port behavior
- Basic service detection concepts
- Reconnaissance techniques used in penetration testing

---

## ⚙️ Features

- Fast TCP port scanning
- Target IP analysis
- Open port detection
- Lightweight and simple execution
- Beginner-friendly structure for learning

---

## 🚀 Usage

```bash
python network-scanner.py <target-ip>
